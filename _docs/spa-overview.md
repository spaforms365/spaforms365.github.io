<code>SPA Form project</code> is a collection of <b>[Knockout components](http://knockoutjs.com/documentation/component-overview.html){:target="_blank"}</b>. Inside of the project, components are organized into hierarchcal structures. Each <code>component</code> basically is a pair of HTML <code>Template</code> and Javascript <code>Model</code>. 
<br/>
At runtime each component in project hierarchy instanciates its Template/Model pair into <code>MVVM runtime model</code>. Altogether, these instances produces <code>Single Page Application</code> powering SPA Form.
